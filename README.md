# goit-cs-hw-02 (ДЗ-2)

Домашнє завдання з нормалізації баз даних та моделювання ER-діаграми.

## Опис завдання
Нормалізація таблиці замовлень від початкової форми до третьої нормальної форми (3НФ) та створення бази даних у MySQL.

## Структура документа hw-02.docx:
- **p1** — початкова таблиця
- **p2** — 1НФ (Перша нормальна форма)
- **p3** — 2НФ (Друга нормальна форма)
- **p4** — 3НФ (Третя нормальна форма)
- **p5** — ER-діаграма
- **p6** — SQL-код (Forward Engineer)
- **p7** — розгорнута схема в Workbench

## Технології
- MySQL Workbench 8.0.31
- MySQL Server 8.4.7

## Результат
Створено 3 таблиці:
- **Client** — клієнти
- **Order** — замовлення
- **Order_details** — деталі замовлень
